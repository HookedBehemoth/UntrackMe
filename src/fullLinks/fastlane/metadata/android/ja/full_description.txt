UntrackMeをデフォルトでTwitter、YouTube、Instagramのリンクを開くように設定すると、Nitter、Invidious、Bibliogramのリンクに変換し、別のアプリのアクションに任せます。

また、GoogleマップのリンクをOpenStreetMapのリンクやGeoURIに変換することもできます。

アプリは短縮URLを検出し、サイトにアクセスする前にフルURLを表示します。

**Liteバージョン:

- Twitter、YouTube、Googleマップのリンクを共有するときに、UntrackMeを使ってリンクを変換してから別のアプリに共有します。

**フルバージョン:

- Lite バージョンのすべての機能に加えて、すべてのリンクのトラッキングパラメータを削除する処理もできます。 訪問前にURLを表示する独自のアプリピッカーを使用します。


Nitterとは:

プライバシーに重点を置いた無料でオープンソースのTwitter代替フロントエンドです。
Github プロジェクト: https://github.com/zedeus/nitter


Invidiousとは:

YouTubeの無料でオープンソースの代替フロントエンドです。
Github プロジェクト: https://github.com/omarroth/invidious


Bibliogramとは:

Instagramの無料でオープンソースの代替フロントエンドです
Github プロジェクト: https://github.com/cloudrac3r/bibliogram


Tedditとは

プライバシーに重点を置いた無料でオープンソースのRedditの代替フロントエンドです。
Github プロジェクト: https://github.com/teddit-net/teddit


Scribeとは:

Mediumの代替フロントエンドです。
ソース: https://git.sr.ht/~edwardloveall/scribe


ウィキレスとは:

プライバシーに重点を置いた無料でオープンソースのWikipediaの代替フロントエンドです。
プロジェクト: https://codeberg.org/org/orenom/Wikiless
