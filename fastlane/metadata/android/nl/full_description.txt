NitterizeMe is standaard ingesteld om Twitter en YouTube links te openen en zal deze omzetten in Nitter en Invidious links en de actie naar een andere app delegeren.

### Wat is Nitter:

Een gratis en open source alternatief Twitter frontend gericht op privacy.
Github project: [/zedeus/nitter](https://github.com/zedeus/nitter)


### Wat is Invidious:

Een gratis en open source alternatieve frontend voor YouTube
Github project: [/omarroth/invidious](https://github.com/omarroth/invidious)
