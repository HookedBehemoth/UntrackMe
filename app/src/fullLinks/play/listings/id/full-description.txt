UntrackMe once set by default to open Twitter, YouTube and Instagram links will transform them into Nitter, Invidious and Bibliogram links and delegate the action to another app.

Ini juga mendukung tautan Google Maps untuk mengubahnya menjadi tautan OpenStreetMap atau Geo URI.

Aplikasi akan mendeteksi URL singkat dan akan menampilkan URL lengkap sebelum mengunjungi situs.

**Lite version:

- When sharing a Twitter, YouTube or Google Maps link just pick up UntrackMe to transform it before sharing it with another app.

**Full version:

- Contains all features of the lite version and it also handle all links to remove tracking parameters. It uses its own app picker that display URLs before visiting them.


Apa itu Nitter:

Front-end Twitter alternatif gratis dan open source yang berfokus pada privasi.
Github proyek: https://github.com/zedeus/nitter


Apa itu Invidious:

Front-end alternatif sumber bebas dan terbuka untuk YouTube
Github proyek: https://github.com/omarroth/invidious


What is Bibliogram:

A free and open source alternative front-end to Instagram
Github proyek: https://github.com/cloudrac3r/bibliogram
